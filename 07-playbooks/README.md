# GTM Playbooks

## Overview

This directory contains tactical playbooks for executing DeepRead's GTM strategy. Each playbook provides step-by-step instructions for common scenarios.

## Available Playbooks

### Sales & Outbound
- **[SDR High-PQL Outreach](./sdr-high-pql-outreach.md)** - How to engage high-value product qualified leads
- **[AE Demo Playbook](./ae-demo-playbook.md)** - Running effective product demos
- **[Enterprise Deal Playbook](./enterprise-deal-playbook.md)** - Closing >$50K contracts

### Customer Success
- **[Onboarding Playbook](./customer-onboarding.md)** - First 30 days after contract signed
- **[Expansion Playbook](./expansion-playbook.md)** - Identifying and closing upsells
- **[Churn Prevention](./churn-prevention.md)** - Saving at-risk customers

### Marketing
- **[Content Creation](./content-creation.md)** - Writing high-converting docs and tutorials
- **[Launch Playbook](./product-launch.md)** - Launching new features to customers

### Operations
- **[Weekly Metrics Review](./weekly-metrics-review.md)** - What to review and when
- **[PQL Scoring Updates](./pql-scoring-updates.md)** - Refining PQL model based on data

## How to Use These Playbooks

1. **Choose the relevant playbook** for your current situation
2. **Follow the steps sequentially** - they're designed to be comprehensive
3. **Customize as needed** - use these as templates, adapt to your context
4. **Track results** - note what works and what doesn't, iterate
5. **Update playbooks** - as you learn, improve these documents

## Playbook Template

When creating new playbooks, use this structure:

```markdown
# [Playbook Name]

## When to Use This Playbook
[Trigger conditions or scenarios]

## Prerequisites
[What you need before starting]

## Steps
[Numbered, actionable steps]

## Success Criteria
[How to know if this worked]

## Common Pitfalls
[Mistakes to avoid]

## Examples
[Real or hypothetical examples]
```
